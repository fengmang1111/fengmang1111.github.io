Demo 演示

技术方案

本 Demo 使用 MediaPipe Hands 框架实现手部 21 个关键点的实时检测与可视化。通过 OpenCV 调用摄像头，利用 MediaPipe 预训练模型对每帧图像进行手部定位与关键点回归，最后将关键点坐标绘制到画面上并连接成骨架线。

环境搭建与问题排查记录
在 Python 3.14 + MediaPipe 0.10.x 环境下，完成了虚拟环境搭建、清华镜像源配置。排查过程中解决了摄像头权限、API 版本兼容等问题，最终成功跑通实时手部检测 Demo。
